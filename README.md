# 🌐 Angular Learning Guide

This repository is a beginner-friendly guide for learning **Angular**, a popular frontend framework by Google used to build dynamic and scalable web applications.

---

## 📌 What is Angular?

Angular is a **TypeScript-based** open-source frontend framework for building **single-page applications (SPAs)**. It offers:

- Component-based architecture
- Built-in routing and HTTP support
- CLI for project scaffolding and automation
- Integration with RxJS for reactive programming
- Tools for unit testing and end-to-end testing

---

## 🧠 Prerequisites

Before you start learning Angular, you should have:

- Basic understanding of **HTML**, **CSS**, and **JavaScript**
- Familiarity with **TypeScript** is helpful, but not mandatory
- Node.js (LTS version) installed
